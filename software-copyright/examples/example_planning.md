## 6. 技术特点摘要

1. 基于 Vue 3 Composition API 的响应式前端架构，支持组件化开发和按需加载
2. 采用 RESTful API 设计规范，前后端完全分离，支持多端接入
3. Redis 缓存热点数据，支持高并发场景下的快速响应
4. JWT Token 双重认证机制，接口级权限控制，保障数据安全
5. UniApp 跨端方案，一套代码同时支持微信小程序、H5 和 App
